# 🎌 AnimeHub

An open-source anime streaming platform built with a full **PRD, Design System, and Tech Architecture** — vibe coded from idea to deployment.

> ⚠️ Currently in active development. Video streaming integration coming in Phase 2.

---

## ✨ Features

- 🎬 Clean, modern dark UI with green accent design system
- 🔍 Browse, search, and filter anime by genre and trending
- 📱 Fully responsive — mobile, tablet, and desktop
- 🎥 Video player UI ready (streaming integration pending)
- 👤 User authentication pages (Signup, Login, Profile)
- 🗂️ Watch history, favorites, and watchlist support
- ⚡ Optimized performance with lazy loading and CDN-ready architecture

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm or pnpm

### Installation

```bash
# Clone the repo
git clone https://github.com/syed-kaif07/animehub.git
cd animehub

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
